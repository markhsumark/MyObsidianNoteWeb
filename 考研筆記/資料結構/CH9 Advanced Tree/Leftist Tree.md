 Leftist Tree

#⭐️⭐️⭐️

**緣由**

|                     | Heap    | Leftist     |
| ------------------- | ------- | ----------- |
| Insert x            | O(logn) | `O(logn)`     |
| Delete-min          | O(logn) | `O(logn)`     |
| **Merge two Heaps** | O(n)    | `O(logn)` |

> 定義：針對每一個internal node x，[shortest](shortest.md)(x->Lchild) >= [shortest](shortest.md)(x->Rchild)。~~左半大於右半~~
