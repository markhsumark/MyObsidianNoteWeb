常見的operator優先表

| operator                      | 優先權 | 結合性 | unary/binary |
| ----------------------------- | ------ | ------ | ------------ |
| 括號配對                      | *高*   |        |              |
| 負號                          | .      |        | nuary        |
| 冪次方(＊＊, $)               | .      | 右結合 | binary       |
| ＊, /                         | .      | 左結合 | binary       |
| +, -                          | .      | 左結合 | binary       |
| 關係(>, <, ＝＝, <=, !=, ...) | .      | 左結合 | binary       |
| 邏輯（否定）                  | .      |        | unary        |
| AND, OR                       | .      | 左結合 | binary       |
| 指定(Assign), =, <-           | *低*   | 右結合 | binary       |
|                               |        |        |              |
